# Installation

Install RoQuery by requiring the module.

=== "Lua"

    ```lua
    local S = require("path/to/RoQuery.lua")
    ```

# API Reference

## RoQuery(selector)
Type
  - Class:Method:constructor

Params
  - selector
    * Type: string
    * Description: The selector used to fetch Instances from the Instance
   * Returns: ElementGroup
